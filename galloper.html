Politique de confidentialité — Galloper

1. Responsable du traitement

Éditeur / Développeur : Sergey Borisenko & Devus Technologies

2. Données collectées

L’application ne collecte pas et n’envoie pas de données personnelles à un serveur exploité par l’éditeur.

  • Stockage local : l’application peut enregistrer le meilleur score localement sur l’appareil (ex. via stockage local) afin de l’afficher ensuite.

3. Publicité, analytics, suivi

  • Pas de publicité intégrée.
  • Pas d’analytics intégré.
  • Pas de suivi publicitaire.

4. Permissions

L’application ne requiert pas de permissions sensibles pour fonctionner, hors permissions techniques standard du système.

5. Liens externes

Un bouton peut ouvrir un lien externe (ex. “Lire un article”). Les sites externes ont leurs propres règles de confidentialité.

6. Enfants / mineurs

Jeu sans compte et sans collecte de données personnelles. Si l’application est destinée aux enfants, toute évolution (pub/analytics) devra respecter les règles applicables et entraîner une mise à jour de cette politique.

7. Sécurité

Les données restent sur l’appareil. Vous pouvez les supprimer via les réglages Android (effacer données) ou en désinstallant l’application.

8. Modifications

Cette politique peut évoluer ; la date de mise à jour sera alors modifiée.
